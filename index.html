Projects

Motion Fitness

With this project, my team and I wanted to develop a tool that would help users to preform safe exercises when they workout either at home or the gym. We designed an application where we used a Xbox 360 kinect and a software we installed called blender. From then we used python to have the kinect track the user so when the user is doing a certain exercise it will let the user know whether he/she is doing it correctly. If they are doing it wrong, like the wrong stance it will highlight red, if they are doing it correctly it will be blue. This was a really fun project and good introduction to python.

Financial Tracker

The goal of the project was to use a certain framework and develop an application to benifit the community. My partner and I decided to make a financial tracker in which we used bootstrap and the Angular framework to develop this application. The application will have categories of your finances and keep track of your expenses. It will also do a bi-weekly report to give tips and other important information to make sure you are financially stable. Although the project is a work in progress, my partner and I will be finishing this app by the end of the year.

Database Housing System

Using Oracle SQL, my team and I decided to make a Housing System for temple students living in dorms. We made the database to organize and how each person would handle different things in this system. We made a database for each part, from RA, to Furniture and made it easier for people who are running these dorms, get a sense of control and how each part for this dorming busisness will be organized. This was a really enjoyable project and although we had some rough patches, we developed a good mock up database that can be used in the future.
